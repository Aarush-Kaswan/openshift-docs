= roxctl cluster delete

Remove Sensor from Central.

.Usage
[source,terminal]
----
$ roxctl cluster delete [flags]
----

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--name string`
|Specify the cluster name to delete.
|===
