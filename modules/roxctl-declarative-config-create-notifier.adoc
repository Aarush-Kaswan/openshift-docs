= roxctl declarative-config create notifier

Create a declarative configuration for a notifier.

.Usage
[source,terminal]
----
$ roxctl declarative-config create notifier [flags]
----

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--name string`
|Specify the name of the notifier.
|===
