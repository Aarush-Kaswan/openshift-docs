= Installing sample build strategies
include::_attributes/common-attributes.adoc[]
:context: installing-sample-build-strategies

toc::[]

You can install a build strategy or cluster build strategy on the {ocp-product-title} cluster. A `ClusterBuildStrategy` resource is available cluster-wide and a `BuildStrategy` resource is available within a namespace.

{builds-title} supports the following build strategies on the linux/amd64 platform:

* `buildpacks-v3-heroku`
* `buildpacks-v3`

{builds-title} supports the following cluster build strategies:

* `buildah`
* `buildkit`
* `buildpacks-v3-heroku`
* `buildpacks-v3`
* `kaniko`
* `ko`
* `source-to-image`

[NOTE]
====
The `buildpacks-v3-heroku`, `buildpacks-v3`, and `source-to-image` strategies are only supported on the linux/amd64 platform. Rest of the cluster build strategies are supported on all the platforms.
====
