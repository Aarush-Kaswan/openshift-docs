= Installing sample build strategies
include::_attributes/common-attributes.adoc[]
:context: installing-sample-build-strategies

toc::[]

You can install a build strategy or cluster build strategy on the {ocp-product-title} cluster. A `ClusterBuildStrategy` resource is available cluster-wide and a `BuildStrategy` resource is available within a namespace.


{builds-shortname} supports the following cluster build strategies:

* `buildah`: Supported on all platforms
* `source-to-image`: Supported on the linux/amd64 platform

[NOTE]
====
The `buildpacks` build strategy is currently in link:https://access.redhat.com/support/offerings/devpreview[Developer Preview].
====
